# Project Start

## Why I am building this
I want to build a local-first AI data engineering platform that can understand any repository and any SQL dialect. The goal is to make complex codebases easier to explore, explain, and analyze.

## What I want this to become
- Repo ingestion
- SQL intelligence
- Search and chat with citations
- File and lineage exploration
- A strong portfolio project showing software engineering, data engineering, and AI application design

## Initial MVP
- FastAPI backend
- Next.js frontend
- Postgres + vector search
- Local repo ingestion
- SQL parsing with graceful fallbacks
